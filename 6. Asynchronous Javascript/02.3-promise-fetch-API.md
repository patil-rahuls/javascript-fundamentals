## Promises & fetch API [ES6]

**Promise is an object that is used as a placeholder for the future result of an asynchronous operation.**

A perfect example of a future value is fetching response from an AJAX call.

```javascript
const result = fetch(".../api/users?page=2");

console.log(result);
// Promise {<pending>}
```

To consume the promise we use/chain `.then()` method. It takes 2 parameters:

- `<callback for fulfilled promise>`,
- `<callback for rejected promise>`

> ```javascript
> result
>   .then(function (response) {
>     // callback for fulfilled promise
> 
>     console.log(response);
>     // Contains 'Body' property which is of type "ReadableStream".
>     // We use json() to read the JSON data.
> 
>     return response.json();
>     // json() is also an asynchronous function and
>     // returns a new Promise.
>   })
>   .then(function (data) {
>     console.log(data);
>   });
> ```
> 
> _Observe how we have chained the promises together instead of nesting them like we do using callbacks._

> Cleaner usable version:
> 
> ```javascript
> const getData = () => {
>   fetch("https://reqres.in/api/users?page=2")
>     .then((response) => response.json())
>     .then((data) => console.log(data));
> };
> ```

_`fetch()` is a lot simpler and minimal to use instead of the traditional AJAX using XMLHTTPREQUEST._

Also we avoid the Callback Hell by using promises.

Example

```javascript
(function () {
  console.log("Fetching Data asynchronously...");
  fetch("https://reqres.in/api/users?page=2")
    .then((response) => {
      console.log("Fetched Data");
      return response.json();
    })
    .then((data) => console.log(data));
})();
// Output:
// Fetching Data asynchronously...
// Fetched Data // Prints after 5 seconds.
// {...}
```

Example: _If we do not pass any callback to a then(), it is ignored and the next chained then() method is executed._

> ```javascript
> var p = new Promise(function (resolve, reject) {
>   resolve("OK");
> });
> p.then()
>   .then()
>   .then(function (data) {
>     console.log(data);
>   });
> 
> // Output
> // "OK"
> ```
> 
> _In the above example, we chained the empty `then()` method twice. But the last non-empty `then()` will be operating on the promise object `p`._
