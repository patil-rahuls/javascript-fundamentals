https://github.com/Unitech/pm2

PM2 is a cluster management solution.
Managing, starting and stopping health.
If any child crashes, this can restart it.

pm2 start index.js -i 0

-i 0 letting pm2 to decide how many instances to create.

Some useful commands:

pm2 monit
pm2 delete index (delete instaneces)

Usually, we SSH into the machine running the PM2 and
execute these commands.
