## Strings - Methods

```javascript
const brand = 'Apple';
const model = 'Macbook Air';

console.log(brand[0]);              //  A
console.log( 'Apple'[0] );          //  A
console.log( 'Apple'[3] );          //  l
console.log( 'Apple'.length );      //  5
```

Just like `Array` constructor, we have a constructor for strings called `String` and we have access to several methods that we can use on strings primitives.

&nbsp;

### `String.prototype.indexOf()`

Returns index of a string from a string

```javascript
brand.indexOf('e');
// 4 (strings are also 0 based)

model.indexOf('Air');
// 8
```

&nbsp;

### `String.prototype.lastIndexOf()`

Returns last Index Of given string from a string

```javascript
brand.lastIndexOf('p');
// 2

'Mississippi'.lastIndexOf('s');
// 6
```
&nbsp;

### `String.prototype.slice()`

Works same as for array

> slice(start_index , end_index):
>
> element at 'end_index' is not included in the returned result.

```javascript
model.slice(1);

model.slice(-1);
```

&nbsp;

### `splice()`

**No, there is no function as String.prototype.splice(), but you can convert string to Array and then use `splice()`.**

```javascript
// Convert the String to Array.
myStr = [...model];

myStr.splice(7);
// Air

myStr.splice(3,7);
// book

myStr.splice(0 , myStr.lastIndexOf(' '));
// Mac

// Convert the Array back to string.
myStr = myStr.join('');
```

IMP: Javascript automatically converts a string primitive into a string Object and operates on it with different methods.

```javascript
console.log(typeof new String("rahul"));
// 'object'
```

On calling slice(), string primitive is converted into object, the method is applied on that object, and then the object is converted back into string primitive and returned.

```javascript
console.log(typeof new String("Rahul").slice(-1));
// 'string'
```
