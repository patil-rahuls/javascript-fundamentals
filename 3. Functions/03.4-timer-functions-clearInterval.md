## Timer Functions - clearInterval()

### `clearInterval()`

Clears the interval function set using setInterval.

> Example: Countdown Timer

```javascript
let max = 5;
const done = setInterval(() => {
  if (max > 0) {
    max--;
    console.log(max);
  }
  // We still need to clear the timer
  // because even after max > 0,
  // it will still keep running
  // forever every 1 second.
}, 1000);

if (max === 0) {
  clearInterval(donex);
}
```
