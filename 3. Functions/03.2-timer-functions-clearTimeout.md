## Timer Functions - clearTimeout()

### `clearTimeout()`

Cancels the execution of a timer.

```javascript
const myTimerFn = setTimeout(() => {
  console.log("Handler fn.");
}, 3000);

if (somecondition) {
  clearTimeout(myTimerFn);
}
```
