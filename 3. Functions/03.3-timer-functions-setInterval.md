## Timer Functions - setInterval()

### `setInterval()`

Calls the callback function for every specified time interval.

```javascript
setInterval(() => {
  const now = new Date();
  console.log(now);
}, 1000);

```
_In the above example, the code will keep printing the date every 1 second to the console._
