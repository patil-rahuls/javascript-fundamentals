## Timer Functions - setTimeout()

### `setTimeout()`

```javascript
setTimeout(() => console.log(`started after 3 secs`), 3000);

console.log(`Waiting...`);
// 'Waiting...'
// 'started after 3 secs'
```

> _In the above example,_
>
> _`setTimeout()` registers the function passed as an argument to > be executed after given milliseconds._
>
> _Then the `'Waiting...'` is printed and whatever code follows, > is executed._
>
> _Meanwhile in the background, the registered function waits for 3000 milliseconds and then it also executes irrespective of where our control is._

&nbsp;

> Example: `setTimeout()` _with parameters and arguments_

```javascript
setTimeout(
  (val1, val2) =>
    console.log(
      `Values passed here are
    ${val1} and ${val2}`,
    ),
  3000,
  "orange",
  "mango",
);
// The arguments 'orange' and 'mango' are
// passed to the function defined
// inside the setTimeout.

console.log(`Waiting...`);
// 'Waiting...'
// 'Values passed here are
//  orange and mango'
```
