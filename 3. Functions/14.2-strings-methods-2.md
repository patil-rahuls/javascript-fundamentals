## Strings - Methods

More Strings methods.

```javascript
model.toLowerCase();

model.toUpperCase();

const name = "rAHUL";

const newName = name[0].toUpperCase() + name.slice(1);
// "RAHUL"
```


### `String.prototype.trim()`

Trims empty spaces at the start and ends of a string.

```javascript
const newStr = myStr.trim();
```

&nbsp;

### `String.prototype.replace()`
Replaces first occurrence of subsrtring.

```javascript
newStr = myStr.replace("to_b_replaced", "replaces_with_this");
```

&nbsp;

### `String.prototype.replaceAll()`

Replaces all occurrences of substring.

```javascript
newStr = myStr.replaceAll(" ", "_");
```

Advantage of using string methods is that, we can chain functions like this.
```javascript
newStr = myStr.replace(" ", "_").replace("$", "Rs.");
```

Alternate to replaceAll using regex.
```javascript
newStr = myStr.replaceAll(/ /g, "_");
// g stands for global.
// Means apply to the entire string.
```

&nbsp;

### `String.prototype.includes()`
Checks if a string contains a substring.

Returns a BOOLEAN
```javascript
return myStr.includes("search_str");
```


&nbsp;

### `String.prototype.startsWith()`

Checks if a string starts with given substring
```javascript
return myStr.startsWith("R");
```

&nbsp;

### `String.prototype.endsWith()`
Checks if a string end with given substring
```javascript
return myStr.endsWith("R");
```


&nbsp;

### `String.prototype.split()`

Explodes a string into an array.
```javascript
const str = "Ubuntu's UI is ACTUALLY slow, Admit it!";
const resultArray = str.split(' ');
```

Remember to use destructuring wherever we get arrays
```javascript
const [firstName, lastName] = 'Rahul Patil'.split(" ");
```


**join()**: implodes an array into a string
```javascript
const newName2 = ['Rahul', 'Patil'.toUpperCase()].join(' ');
// Rahul PATIL

const newName3 = ['Rahul', 'Patil'].join('-');
// Rahul-Patil
```


&nbsp;

### `String.prototype.padStart()` &

Padds a string with given string in the begining
> Syntax
>
> padStart(fixedLength, char)

Padding changes the length of the string by adding extra string specified by the user at the end or starting.

```javascript
const lol = "Barney Stinson";
newStr = lol.padStart(15, '+');

console.log(newStr);
// "+Barney Stinson" (Total length become 15)

newStr = lol.padStart(25, '+');

console.log(newStr);
// "+++++++++++Barney Stinson"  (Total length - 25)
```

&nbsp;

### `String.prototype.padEnd()`

Padds a string with given string at the end.
> Syntax
>
> padEnd(fixedLength, char)

```javascript
newStr = go.padEnd(25, '-');
```

> Real world example: Mask the debit/credit card numbers.

```javascript
const maskCreditCard = function(number){
  let given = '' + number;
  // parameter converted to a string. Now we
  // can pass a number as well as a string
  // to this function.

  // Get the last 4 digits and length.
  const last = given.slice(-4);

  // start from last 4th char.
  const len = given.length;

  const newVal = last.padStart(len, '*');

  console.log( newVal );
}

maskCreditCard(5674657210978324);
// will accept number type.

maskCreditCard('1200987564738523');
// will accept string too.
```

### `String.prototype.repeat()`
Repeats strings given number of times

> Real world Example: Useful to show icons like ratings, stars etc.

```javascript
let rating = 4;
console.log(`You have got
  ${rating} rating \n ${'🌟'.repeat(rating)}`);
```
