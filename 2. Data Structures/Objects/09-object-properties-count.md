## Data Structures > Object

### Size of an Object using `Object.keys().length`

> Example: Get size of an Object
```javascript
const myObj = {
  firstName : 'Rahul',
  lastName : 'Patil',
  age : 2049-2024,
};

// Size of an object is the number of properties it holds.
let size = Object.keys(myObj).length;

console.log(size);
// 3
```
