## Data Structures > Object

### Objects Properties Count using `Object.keys().length`

Consider this object as an example.

```javascript
const myObj = {
  firstName : 'Rahul',
  lastName : 'Patil',
  age : 2049-2024,
};

// Size of an object is the number of properties it holds.
let size = Object.keys(myObj).length;

console.log(size);
// 3
```
