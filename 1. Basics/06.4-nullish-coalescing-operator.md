## Nullish Coalescing Operator `??`

> It works same as `||` except that it does not consider the following as falsy values:
>
> - **_`0`_**
> - **_`''`_**
> - **_`Infinity`_**
> - **_`NaN`_**

Nullish means only `null` and `undefined` will be considered falsy.

> Example 1

```javascript
const usersCount = 0;

const maxUsers = usersCount || 10;

console.log(maxUsers);
// 10

// When we want to consider '0'
// as a valid value for 'maxUsers'
// Solution : use nullish/coalesce operator.
const maxUsersCount = usersCount ?? 10;

console.log(maxUsersCount);
// 0
```

&nbsp;

> Example 2

```javascript
const isDeleted = "";

const result = isDeleted ?? 1;

console.log(result);
// ''
```

&nbsp;

> Example 3: Always use `||` to handle `Infinity` and `NaN`

_Using `??` with `Infinity` and `NaN` is not worth to use, because it will be returning `Infinity` / `NaN`._

```javascript
const minUsers = NaN;
const totalUsers = minUsers ?? 23;
console.log(totalUsers);
// NaN

const users = 1 / 0;
const totalUsrs = users ?? 0;
console.log(totalUsrs);
// Infinity

const minUsrs = minUsers || 23;
console.log(minUsrs);
// 23
```
