## Nested Destructuring Assignment [ES6]

Unpacks and assigns values from arrays, or properties from objects into distinct variables.

### 1. Nested Arrays

```javascript
const arr = [2, 4, 6, [1, 3, 5]];
const [i , , , j]  = arr;
console.log(i, j);
// 2   [1, 3, 5] ;

const [, x, ,  [, , y]] = arr;
console.log(x , y);
// 4   5
```

### 2. Nested Objects

```javascript
const user = {
  name: "Rahul",
  age: 25,
  city: "Pune",
  workingHours : {
    weekdays: {
      start: 10,
      end: 19,
    },
    weekends: {
      start: 12,
      end: 16,
    }
  }
};
const {
  workingHours: {
    weekdays: {
      start: shiftStartsAt,
      end: shiftEndsAt
    }
  }
} = user;
console.log(shiftStartsAt, shiftEndsAt);
// 10  19
```

&nbsp;

When destructing nested objects, we MUST use correct property names _(i.e. we used 'weekdays', 'start', 'end', 'workingHours' in the above example correctly)_

If we (mistakenly) specify an incorrect property while destructuring, we get error.


```javascript
// Consider the same 'user' object from previous example
const {
  workingHours: {
    monday: {    // Incorrect property
      start: shiftStartsAt,
      end: shiftEndsAt
    }
  }
} = user;
console.log(shiftStartsAt, shiftEndsAt);
// Uncaught TypeError: Cannot read properties of undefined (reading 'start')
// Because, JavaScript couldn't find the object 'monday' inside the 'workingHours' object while destructuring it.
```

> Destructuring function params

Calling an object method which takes a destructured object as its parameter.

```javascript
const user = {
  name: "Rahul",
  age: 25,
  city: "Pune",
  notify({ name, msg }) { // Destructuring params
    console.log(`${name} sent: ${msg}`);
  }
};
const payload = {
  name: 'Admin',
  msg: 'Hello Rahul!'
};
user.notify(payload);
// We called the method 'notify' by passing an object 'payload'.
// The object 'payload'(parameter) is destructured in the method definition.
```
