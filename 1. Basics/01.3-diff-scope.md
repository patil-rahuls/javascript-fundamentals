## `let`, `const` & `var` - Scope

> **_`let` and `const` variables are `Block scoped`._**
>
> **_`var` variables are `Function scoped`._**

_**`Scope`** refers to the visibility or accessibility of variables, functions and objects within different parts of a program. It determines where in the program a particular variable or a function can be accessed and manipulated._

_A block is any code within curly braces `{ }`, including functions, loops and conditional statements._

`let` and `const` variables are scoped to the immediate enclosing block denoted by `{ }` _(hence the ***`Block scope`***)_.

`var` variables are scoped to the immediate function body `function() {...}` _(hence the ***`Function  scope`***)_.

> Example 1
>
> ```javascript
> function run() {
>   // Function scope
>   var foo = "Foo";
>   let bar = "Bar";
>
>   console.log(foo, bar);
>   // 'Foo Bar'
>
>   {
>     // Block scope
>     var moo = "Mooo";
>     let baz = "Bazz";
>
>     console.log(moo, baz);
>     // 'Mooo Bazz'
>   }
>
>   console.log(moo);
>   // 'Mooo'
>
>   console.log(baz);
>   // 'Uncaught ReferenceError: baz is not defined'
> }
>
> run();
> ```

&nbsp;

> Example 2
>
> ```javascript
> var x = "global scope";
>
> function foo() {
>   var x = "function scope";
>   console.log(x);
> }
>
> {
>   let x = "block scope";
>   // 'x' is only accessible in
>   // this enclosing block
> }
>
> foo();
> // 'function scope'
>
> console.log(x);
> // 'global scope'
> ```

&nbsp;

> Example 3: **_Scope changes when a variable is declared without `let`, `var` or `const` keyword._**
>
> _In the following code, `b` is not declared using any of the `let`, `const` or `var` keywords and by default becomes `globally scoped`._
>
> ```javascript
> {
>   let a = 9;
>   b = 8; // Becomes globally scoped
> }
>
> console.log(a);
> // 'ReferenceError: a is not defined'
> // ('let' variable is block scoped)
>
> console.log(b);
> // '8'
> // ('b' is globally scoped)
> ```
>
> _`Globally scoped` means the variable is accessible by the global object._
>
> _In Browser, we can access the variable `b` using the global object - `window`._
>
> ```javascript
> console.log(window.b);
> ```
>
> _In Node JS runtime, we can access `b` using `global` object. (the name itself is `global`)_
>
> ```javascript
> console.log(global.b);
> ```
