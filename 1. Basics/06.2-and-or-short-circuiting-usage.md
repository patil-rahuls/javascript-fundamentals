## Short Circuiting `&&` and `||`

The operators `||` and `&&` do not always return boolean.

> Example 1: Default value if first operand is falsy

```javascript
const myObj = {};

const val1 = myObj.property1 ? myObj.property1 : 10;
```

> _Better Solution: We can use short circuiting the `OR` operator to shorten the above code._
>
> ```javascript
> const val2 = myObj.property1 || 10;
> // If first operand is falsy,
> // we will have default value
> // 10 assigned to val2.
> ```

&nbsp;

> Example 2: Calling a function if it exists

```javascript
const calcSq = function (val) {
  return val ** 2; // val^2
};

calcSq && calcSq(34);
// if calcSq() function exists,
// then the second expression
// calcSq(34) will be executed.
```

&nbsp;

For variables, use `||`

    const age = myAge || 26;

For function calls, use `&&`

    calcSq && calcSq(34);
